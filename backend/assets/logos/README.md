# Logo Assets

Place your company logo PNG files here:

- `lakeb2b.png` - LakeB2B logo
- `champions.png` - Champions Group logo
- `ampliz.png` - Ampliz logo

## Requirements

- **Format:** PNG with transparency
- **Size:** Recommended 200-400px width, height proportional
- **Background:** Transparent (for clean overlay)

## Usage

These logos are overlaid on videos to replace removed watermarks.
The logo is automatically scaled to ~120px width when applied.
